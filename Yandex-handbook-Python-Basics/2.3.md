F. НОД
```python
x, y = int(input()), int(input())
x_max = max(x, y)
y_min = min(x, y)
r_k = x_max - (y_min * (x_max // y_min))
if r_k == 0:
    print(y_min)
else:
    while r_k != 0:
        r_i = r_k
        x_max = y_min
        y_min = r_k
        r_k = x_max - (y_min * (x_max // y_min))
    print(r_i)

#Другое решение: короче и менее затратно по времени и ресурсам.
x, y = int(input()), int(input())
while y:
    x, y = y, x % y
print(x)
```
